## Long Description
NikNote is a revolutionary AI-powered learning and handwriting platform built specifically for Indian students. Founded by Nikhil Jatav, NikNote combines the power of Notion, ChatGPT, Perplexity, NotebookLM, and GoodNotes into one free app.

**Key Features:**
- ✍️ **16+ Handwriting Styles**: Convert typed text into realistic handwritten notes with natural imperfections (baseline jitter, stroke variation)
- 🧠 **AI Teacher (Hindi + English)**: Ask any question — Newton's laws, Photosynthesis, Algebra — get instant explanations in Hindi & English
- 📝 **Quiz Generator**: Enter any topic, get quiz questions instantly. CBSE, JEE, NEET pattern
- 🃏 **Flashcard Maker**: Create flashcards for revision. AI generates them automatically
- 🗺️ **Mind Maps**: Visual learning for complex topics
- 📄 **14 Paper Styles**: Ruled, college-ruled, graph, dotted, plain — like your school notebook
- 🎨 **12 Ink Colors**: Blue, black, red, green, and more + custom color picker
- 📥 **PDF Export (300 DPI)**: Print-ready PDFs, A4 size, perfect for assignments
- 📱 **Works on Phone & Laptop**: Fully responsive PWA, install like an app
- 🔌 **Offline Mode**: 28+ subjects work without internet
- 📲 **WhatsApp Share**: One-click share with classmates
- 🎓 **28+ Built-in Subjects**: Physics, Chemistry, Biology, Math, History, Constitution, English Grammar, Programming — instant notes without API key
- 🔐 **Secure**: Data encrypted, Supabase backend, GDPR ready

**Target Audience**: Indian students (Class 5-12, JEE/NEET aspirants, college students, UPSC aspirants)

**Pricing**: Free to start. Student Pro ₹99/month. Lifetime ₹999.

**Tech Stack**: React, TypeScript, Supabase, Tailwind CSS, Vite, PWA

**Made in India by Nikhil Jatav** 🇮🇳
